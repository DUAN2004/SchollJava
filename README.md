基于JavaSwing+Mysql完成的英语单词管理系统，使用Maven管理整个项目

实现的功能

（1）单词的增删查改，当前页面超过20个单词，自动生成分页按钮

（2）例句的增删查改，当前页面超过20个例句，自动生成分页按钮

（3）单词和例句的全文检索

（4）调用百度翻译API,实现单词和例句的翻译（需要自己去百度开放平台申请相应的key才能使用！）
